Building My First Visualization

Let's learn how to create a custom visualization with our Javascript Visualization SDK and embed it to your dashboard. The tutorial is divided into several logical parts based on steps that you have to accomplish. You can also go directly to the examples folder in your downloaded library/repository to check all available prebuilt examples.

Follow the main three steps to deliver a custom visualization:

1. Get the data from the GoodData Platform

Getting the data from the GoodData Platform is a critical part. Since this step will be the same for every data visualization you have to know the structure of the data that is extracted from GoodData.

Sometimes, when the data visualization library requires more complicated input, you have to transform your data to fit this requirements


Learn the details >>

2. Set up the visualization

The second step is always to set up the visualization itself. It is always unique and depends on which visualization library you are using.

The best way to learn how to set up individual Visualization is to explore our example library below. Or definitely read other related D3.js or dc.js tutorials.

Learn how to setup (d3.js) >>

3. Embed the visualization to the dashboard

The final step is to embed the visualization to your GoodData Project.

This task is easy, you can embed the custom visualization in the same way as any other html content. Anyway, since it is a javascript running outside of the GoodData Platform, there are several limitations and requirements. We strongly recommend you to go through the details.

Learn how to embed the visualization >>

That's it! You are ready!

You can know check out all examples we have prepared for you. You can play around or even create new examples and share it with the community.
